반복문 안에 선택문

for문이나 while문과 같은 반복 구조 내부에 if문과 같은 선택 구조가 중첩되어 있는 구조이다.
반복문 안에 선택문 프로그램 예시
random 모듈
  동영상의 프로그램에서 random 모듈을 사용하는데, 모듈이란 이미 정의된 변수나 함수의 집합을 말한다. random은 무작위로 생성한 숫자가 필요할 때 사용하는 외부 모듈이며, 외부 모듈을 사용하기 위해 예약어 import를 사용한다.
import random
number = random.randint(1, 100)
  random 모듈에는 무작위 숫자인 난수를 생성하는 다양한 함수들이 정의되어 있다. 난수 생성 함수 중 일부를 사용하는 예시를 살펴보자.
난수 생성 함수 중 일부를 사용하는 예시
사용 예시 난수의 범위
random.random() 0~1 사이의 실수
random.randint(1, 100) 1~100 사이의 정수
random.sample(range(10), 3) 0~9 중 3개의 숫자
