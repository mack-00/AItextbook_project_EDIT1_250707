사용자 정의 함수

  프로그램을 작성하다 보면 특정 기능을 반복하여 처리하는 경우가 있다. 이때 자주 사용하는 기능을 함수로 정의해서 사용할 수 있다. 함수를 사용하면 함수 안에서 처리되는 명령어를 몰라도 특정 기능을 사용할 수 있는데, 특히 사용자가 직접 작성해서 사용하는 함수를 사용자 정의 함수라고 한다.
사용자 정의 함수 작성법
파이썬 코드 작성법
함수 정의
def 함수명(매개 변수):
명령1
명령2
매개 변수는 함수를 호출할 때 넘겨주는 값을 전달받기 위해 사용하는 변수이다. 전달받을 값이 없다면 생략할 수 있고, 여러 개의 값을 전달받는다면 필요한 만큼 매개 변수를 추가할 수 있다.
return 반환값
반환값은 함수를 처리한 결과를 호출한 쪽으로 되돌려주는 값이다. 반환값이 없다면 생략할 수 있다.
함수 호출
함수명(인수):
함수명을 호출해서 함수에 정의된 기능을 사용할 수 있다. 인수는 함수를 호출할 때 넘겨주는 값이다.
