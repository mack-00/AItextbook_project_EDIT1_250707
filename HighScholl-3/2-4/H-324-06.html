if~elif~else문

if~elif~else문은 선택 구조의 하나로, 여러 경우의 조건을 확인하고 싶을 때 사용하며, 확인하고 싶은 조건의 수만큼 if 뒤에 elif를 추가한다.
if~elif~else문의 작성법
순서도 표현
조건1
참
명령1
거짓
조건2
참
명령2
거짓
조건3
참
명령3
거짓
명령4
파이썬 코드 작성법
if 조건1:
명령1
elif 조건2:
명령2
elif는 if문 없이 사용할 수 없고, 조건1이 거짓일 때 elif의 조건2를 확인한다.
elif 조건3:
명령3
elif는 필요한 조건만큼 추가하여 사용할 수 있다.
else:
명령4
조건1, 2, 3이 모두 거짓일 때 명령4가 실행된다. 명령4에 해당하는 작업이 없다면, else문은 생략 가능하다.
if~elif~else문 프로그램 예시
elif

end 옵션
  동영상의 프로그램에서 print( ) 함수에 end 옵션을 사용하였다. end는 출력된 문자열 끝에 무엇이 올지를 결정하는 옵션으로, 특별히 작성하지 않으면 '\n'(줄 바꿈)을 기본값으로 사용한다. 만약 end 옵션을 ''으로 수정하면, 다음 출력이 줄 바꿈 없이 보이게 된다.
end 옵션 사용 예시
  • 1
    1print('수박', end='') 2print('이 좋아.')
    수박이 좋아.
  • 2
    1print('수박이', end = ' ') 2print('좋아.')
    수박이 좋아.
  • 3
    1print('수박이', end = ' 너무 ') 2print('좋아.')
    수박이 너무 좋아.
