함수의 종류

  프로그램을 작성하면서 자주 사용하는 기능을 함수로 정의해서 사용할 수 있다. 파이썬에서는 함수를 내장 함수, 외장 함수, 사용자 정의 함수로 분류한다.
  • 내장 함수
  • 외장 함수
  • 사용자 정의 함수
내장 함수는 파이썬에서 기본적으로 제공하는 함수로, 빌트인(built-in) 함수라고도 한다.
내장 함수 예시
내장 함수와 설명
내장 함수 설명
print() 문자열을 출력한다.
input() 문자열로 입력을 받는다.
str(n) 숫자 n을 문자열 형태로 반환한다.
abs(n) 숫자 n의 절댓값을 반환한다.
divmod(m, n) 숫자 m을 n으로 나눈 몫과 나머지를 반환한다.
max(문자열 또는 컨테이너형 변수) 문자열이나 컨테이너형 변수의 요소들 중 최댓값을 반환한다.
  외장 함수는 외부에서 작성한 함수로, 외부 모듈이나 패키지에서 정의되어 있는 함수이다. 외장 함수를 사용하기 위해서는 외부 모듈이나 패키지를 불러와야 하며, 이때 예약어인 import를 사용한다.

외장함수와 설명
외장 함수 설명
randint(m, n)

plot()
  • • matplotlib.pyplot 모듈에 정의된 함수
  • • 선그래프를 출력한다.
plot( ) 예시
# as는 모듈에 별칭을 정하는 예약어이다. import matplotlib.pyplot as plt # y축의 1, 4, 2, 7 값을 연결하여 선그래프를 출력한다. plt.plot([1, 4, 2, 7])
사용자 정의 함수는 내장 함수나 외장 함수에서 제공하지 않는 기능을 프로그램 개발자가 직접 작성하는 함수이다.
사용자 정의 함수 예시
1def total(number): 2sum = 0 3for i in range(1, number + 1): 4sum +=i 5return sum
1부터 number까지 정수의 합을 구하는 사용자 정의 함수
1def my_print(text): 2print('[Warning]', text)
내장 함수 print( )를 이용하여 '[Warning]'을
자동으로 붙여주는 사용자 정의 함수
함수의 장점
함수에는 다음과 같은 장점이 있다.
  • 동일한 기능이 필요한 여러 곳에서 다시 사용할 수 있어 코드의 중복이 줄어든다.
  • 특정 기능을 변경해야 할 때 해당 함수만 수정하면 되기 때문에 편리하다.
  • 의미 있는 단위로 분할하기 때문에 코드가 간결하고 함수 내부를 알지 못해도
    기능을 사용할 수 있다.
