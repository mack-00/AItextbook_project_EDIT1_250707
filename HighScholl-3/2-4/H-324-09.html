range( ) 함수

range( ) 함수는 for문과 함께 자주 사용되는 함수로, 특정 구간의 정수 범위를 만들어 준다.
range( ) 함수의 사용법
range(시작,끝,간격)
  • 시작하는 숫자이며, 생략하면 시작 범위를 0으로 설정한다.
  • 반복이 끝나는 숫자이며, '끝-1'까지로 범위를 설정한다. 생략할 수 없다.
  • 시작 숫자부터 간격 숫자만큼 증가하는 범위를 만든다. 생략하면 간격 크기를 1로 설정한다.
range( ) 함수 프로그램 예시
range() 함수의 변숫값 생성 예시
변숫값 생성 예시 변수 i의 범위
for i in range(0, 10, 2) 0, 2, 4, 6, 8
for i in range(5, 10) 5, 6, 7, 8, 9
for i in range(5) 0, 1, 2, 3, 4
for i in range(5, 0, -1) 5, 4, 3, 2, 1
