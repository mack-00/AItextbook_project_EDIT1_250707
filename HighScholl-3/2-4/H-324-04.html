if문

if문은 선택 구조의 하나로, 조건이 참이라면 if문 내부의 명령을 실행하고 조건이 거짓이라면 if문 이후의 명령을 실행한다.
if문의 작성법
순서도 표현
조건1
참
명령1
명령2
거짓
명령3
파이썬 코드 작성법
if 조건1:
if 뒤에 흐름을 제어할 조건이 오게 되며, 그 뒤에 콜론(:)을 붙여 준다.
명령1
if문 내부의 명령들은 들여쓰기를 해야 하며, if문의 조건이 참일 때 실행된다.
명령2
if문 내부의 명령들은 반드시 들여쓰기
위치가 같아야 한다.
명령3
들여쓰기를 하지 않으면 if문이 끝난다.
if문 프로그램 예시
비교 연산자
  동영상의 프로그램에서 다음과 같은 조건식을 작성하였다. 조건식이란 주어진 조건이나 규칙에 따라 계산하여 그 결과가 참 또는 거짓의 값을 갖는 식을 말한다. 조건식에는 비교 연산자를 사용할 수 있다.
if bus == 'y' :
파이썬에서 사용하는 비교 연산자는 아래 표와 같다.
비교 연산자 설명 예시
== 좌항과 우항이 같은지 비교 score == 80 → 점수가 80과 같으면 참
!= 좌항과 우항이 다른지 비교 score != 100 → 점수가 100점이 아니면 참
> 좌항이 우항보다 큰지 비교(초과) score > 80 → 점수가 80 초과이면 참
>= 좌항이 우항보다 크거나 같은지 비교(이상) score >= 80→ 점수가 80 이상이면 참
< 좌항이 우항보다 작은지 비교(미만) score< 80 → 점수가 80 미만이면 참
<= 좌항이 우항보다 작거나 같은지 비교(이하) score<= 80 → 점수가 80 이하이면 참
