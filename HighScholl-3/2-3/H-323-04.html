1차원 데이터의 표현

  1차원 데이터를 파이썬으로 구현할 때 여러 개의 값을 담는 컨테이너형을 사용해 표현한다. 순서가 있는 컨테이너형으로는 리스트가 있고, 순서가 없는 컨테이너형으로는 딕셔너리가 있다.
  • 리스트
  • 딕셔너리
  리스트는 [ ](대괄호) 안에 값을 ‘ , ’(콤마)로 구분해 저장한다. 리스트의 값에 0번부터 1씩 커지는 연속적인 숫자인
인덱스를 부여한다. 인덱스를 이용해서 인덱스 위치에 저장된 데이터를 참조하거나 수정할 수 있다.
  • 리스트명 = [값1, 값2, 값3]
    1차원 리스트의 생성
    1차원 리스트를 생성하는 방법이다. 기억 장소에 리스트의 값이 연달아 저장된다.
    저장된 구조
    • 값1
    • 값2
    • 값3
    • 0
    • 1
    • 2
    인덱스(index)
  • 변수 = 리스트명[인덱스]
    1차원 리스트의 참조
    1차원 리스트에 저장된 데이터를 참조하는 방법이다. 인덱스 위치에 저장된 값을 읽어온다.
    참조 예시
    x = 리스트명[0]
    인덱스 0 위치에 저장된 값1을 읽어 변수 x에 저장한다.
  • 리스트명[인덱스] = 값4
    1차원 리스트의 수정
    1차원 리스트에 저장된 데이터를 수정하는 방법이다. 인덱스 위치에 새로운 값을 저장한다.
    수정 예시
    리스트명[2] = 값4
    인덱스 2 위치에 값4를 저장한다.
    • 값1
    • 값2
    • 값4
  • 리스트명.append(값5)
    1차원 리스트에 값 추가
    리스트에 새로운 값을 추가할 때는 append() 함수를 사용한다.
    추가 예시
    리스트명.append(값5)
    리스트의 마지막에 데이터가 추가된다.
    • 값1
    • 값2
    • 값4
    • 값5
  딕셔너리는 { }(중괄호) 안에 키(key):값(value) 쌍을 저장한다. 딕셔너리는 사전에서 단어를 찾듯이 키를 사용해 값을 가져올 수 있다. 단, 딕셔너리에 중복된 키는 사용할 수 없고, 중복된 키를 입력하면 기존 키의 값이 수정된다.
  • 딕셔너리명 = {키1:값1, 키2:값2}
    1차원 딕셔너리의 생성
    1차원 딕셔너리를 생성하는 방법이다. 기억 장소에 딕셔너리의 키와 값이 연달아 저장된다.
    저장된 구조
    • 키1 값1
    • 키1 값2
  • 변수 = 딕셔너리명[키]
    1차원 딕셔너리의 참조
    1차원 딕셔너리에 저장된 데이터를
    참조하는 방법이다. 키와 한 쌍인 값을 읽어온다.
    참조 예시
    x = 딕셔너리명[키1]
    키1과 한 쌍인 값1을 읽어 변수 x에 저장한다.
  • 딕셔너리명[키2] = 값3
    1차원 딕셔너리의 수정
    1차원 딕셔너리에 저장된 데이터를
    수정하는 방법이다. 키와 한 쌍인 값
    대신 새로운 값을 저장한다.
    수정 예시
    딕셔너리[키2] = 값3
    키2와 한 쌍인 값2 대신 값3을
    저장한다.
    • 키1 값1
    • 키2 값3
  • 딕셔너리명[키4] = 값4
    1차원 딕셔너리에 데이터 추가
    1차원 딕셔너리에 새로운 데이터를 추가하는 방법이다. 딕셔너리에 없는 키와 값을 저장
    한다.
    추가 예시
    딕셔너리[키4] = 값4
    키4와 값4를 추가로저장한다.
    • 키1 값1
    • 키2 값3
    • 키4 값4
