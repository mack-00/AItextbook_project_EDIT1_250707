2차원 데이터의 표현

  2차원 데이터는 1차원 데이터를 모은 것이다. 만약 우리 반 학생들의 상의 치수뿐 아니라 신발 치수, 하의 치수도 함께 모은다면 2차원 데이터를 만들 수 있다. 리스트와 딕셔너리의 2차원 데이터 표현 방식을 비교해 보자.
  • 리스트
  • 딕셔너리
2차원 리스트는 리스트에 리스트를 넣어서 만들 수 있으며, 리스트에 포함된 각 리스트는 콤마로 구분한다.
  • 리스트명 = [[값1, 값2, 값3],[값4, 값5, 값6]]
    2차원 리스트 생성
    2차원 리스트를 생성하는 방법이다.
    데이터가 표 형태로 저장된다고 생각하면 이해하기 편리하다. 가로 방향으로 나열된 데이터를 행(row), 세로 방향으로 나열된 데이터를 열(column)이라고 한다.
    저장된 구조
    가로 방향으로 데이터를 행(row), 세로 방향 데이터를 열(column)
    값1 값2 값3
    값4 값5 값6
    행
    열
  • 변수 = 리스트명[행번호][열번호]
    2차원 리스트 참조
    2차원 리스트에 저장된 데이터를 참조하는 방법이다. 행과 열 인덱스를 모두 사용하여 저장된 값을 읽어온다. 행/열 인덱스는 1차원 리스트와 같이 0으로 시작한다.
    참조 예시
    x = 리스트명[0][0]
    0행, 0열의 값1을 읽어 변수 x에
    저장한다.
  • 리스트명[행번호][열번호] = 값7
    2차원 리스트 수정
    리스트에 저장된 데이터를 수정하는 방법이다. 행과 열 인덱스 위치에 새로운 값을 저장한다.
    수정 예시
    리스트명[1][2] = 값7
    1행, 2열의 위치에 값7을 저장한다.
    1행, 2열의 위치에 값7을 저장
    값1 값2 값3
    값4 값5 값7
  2차원 딕셔너리는 딕셔너리 안에 딕셔너리가 들어 있는 형태이다. 딕셔너리에서 중복된 키는 사용할 수 없지만, 내부 딕셔너리의 키에서는 중복하여 사용할 수 있다.
  • 딕셔너리 = {키1:{키A:값A, 키B:값B},
      키2:{키A:값A, 키B:값}}
    2차원 딕셔너리의 생성
    2차원 딕셔너리를 생성하는 방법이다. 데이터가 표 형태로 저장된다고 생각하면 이해하기 편리하다.
    저장된 구조
    키1
    • 키A 값A
    • 키B 값B
    키2
    • 키A 값A
    • 키B 값B
  • 변수 = 딕셔너리명[외부키][내부키]
    2차원 딕셔너리의 참조
    2차원 딕셔너리에 저장된 데이터를 참조하는 방법이다. 외부 딕셔너리의 키와 내부 딕셔너리의 키를 모두 사용하여 저장된 값을 읽어온다.
    참조 예시
    x = 딕셔너리명[키1][키A]
    키1의 내부 딕셔너리에서 키A와
    한 쌍인 값A를 읽어 변수 x에 저장한다.
  • 딕셔너리명[행번호][열번호] = 값7
    2차원 딕셔너리의 수정
    2차원 딕셔너리에 저장된 데이터를 수정하는 방법이다. 내부 딕셔너리의 키와 한 쌍인 값 대신 새로운 값을 저장한다.
    수정 예시
    딕셔너리명[키2][키B] = 값C
    키2의 내부 딕셔너리에서 키B와 한 쌍인 값으로 값C를 저장한다.
    키1
    • 키A 값A
    • 키B 값B
    키2
    • 키A 값A
    • 키B 값C
try except (예외 처리)
  프로그램을 실행하는 도중 오류가 발생하면, 보통 실행이 중단되고 오류 메시지가 출력된다. 프로그램의 실행이 중단되는 것을 방지하기 위해 예외 처리 구문인 try except를 사용한다. 실행할 코드가 두 줄 이상이면 try:와 except: 다음 줄에 들여쓰기를 해서 여러 줄의 코드를 각 구문 안에 포함시킨다.
예외 처리 코드 작성법
try: 실행할 코드
except: 오류가 발생했을 때 처리할 코드
또는
try: 실행할 코드1 실행할 코드2 except: 오류가 발생했을 때 처리할 코드1 오류가 발생했을 때 처리할 코드2
예외 처리 예시
· · · try:   num = int(input("숫자를 입력하세요: "))   print("결과:", 10 / num) except:   print("오류가 발생했습니다.") · · ·
0 또는 숫자가 아닌 값으로 '10 / num'을 실행하면 오류가 발생하지만, 예외 처리를 하면 프로그램이 중단되지 않고 계속 진행된다.
