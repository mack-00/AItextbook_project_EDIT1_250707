정렬 알고리즘의 장단점 비교

버블 정렬, 삽입 정렬, 퀵 정렬의 장점과 단점을 비교해 보자.
버블 정렬, 삽입 정렬, 퀵 정렬의 장점과 단점
알고리즘 장점 단점
버블 정렬
  • 구현이 매우 간단함.
  • 추가적인 메모리 사용이 적음.
  • 수행 시간이 O(n2)로 많이 소요됨.
  • 데이터가 많아질수록 비효율적임.
삽입 정렬
  • 대부분 정렬된 데이터에 대해 수행 시간이 매우 적게 소요됨.
  • 데이터가 적은 경우 효율적임.
  • 추가적인 메모리가 거의 필요 없음.
  • 최악의 경우 수행 시간이 O(n2)로 많이 소요됨.
  • 데이터가 많아질수록 비효율적임.
퀵 정렬
  • 평균적으로 수행 시간이 매우 적게 소요됨.
  • 대부분의 경우 효율적임.
  • 데이터가 많은 경우 적합함.
  • 추가적인 메모리가 거의 필요 없음.
  • 피벗을 잘못 선택할 경우 수행 시간이 O(n2)까지 증가함.
