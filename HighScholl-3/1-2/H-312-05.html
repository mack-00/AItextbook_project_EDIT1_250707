삽입 정렬

  삽입 정렬이란 연속된 데이터를 정렬된 영역과 그렇지 않은 영역으로 나누고, 정렬되지 않은 영역의 요소를 정렬 기준에 맞게 정렬된 영역으로 한 개씩 삽입하는 방식의 정렬 알고리즘이다.
삽입 정렬의 수행 과정
삽입 정렬의 수행 시간
모든 요소가 정렬된 최선의 경우에는 n -1번의 비교만으로 정렬이 완료되고, 모든 요소가 역순으로 정렬된 최악의 경우에는 비교와 이동 모두 번 수행된다. 삽입 정렬 알고리즘은 요소가 대부분 정렬된 경우에 매우 효율적이다.
삽입 정렬의 수행 시간
최선의 경우 최악의 경우
비교 횟수 n-1 비교 횟수
교환 횟수 0 교환 횟수
수행 시간 O(n) 수행 시간 O(n2)
삽입 정렬 예시
카드 게임 중 정렬되지 않은 영역인 오른손에 있는 카드들을 정렬된 영역인 왼손으로 한 장씩 옮기면서 정렬하는 것은 삽입 정렬을 사용하는 사례에 해당한다.
