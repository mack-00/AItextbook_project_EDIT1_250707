파일입출력

  프로그램을 작성할 때 파일로 저장된 대량의 데이터를 읽어 오거나 다음에 다시 사용하기 위해 결괏값을 파일로 저장해야 하는 경우가 있다. 이처럼 파일에 데이터를 읽고 쓰는 것을 파일입출력이라 한다.
파일입출력은 다음과 같은 순서로 진행된다.
파일 처리 과정
파일 열기
  • 설명

    파일의 데이터를 읽거나 쓰기 위해 파일을 연다.

  • 사용
    하는
    함수

    open()

파일 처리
  • 설명

    파일의 데이터를 읽거나
    파일에 데이터를 쓴다.

  • 사용
    하는
    함수
    • 읽기

      read( )
      readline( )
      readlines( )

    • 쓰기

      write( )

파일 닫기
  • 설명

    파일 처리를 완료하고 파일을 닫는다.

  • 사용
    하는
    함수

    close()
