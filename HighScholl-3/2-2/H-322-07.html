파일의 데이터를 처리하기 위하여 우선 open( ) 함수를 사용해 파일을 연다.
파일 변수 = open( 파일명 ,  파일입출력 모드 )

보충 자료
파일입출력 모드 특징
r(읽기) 파일의 데이터를 읽는다.
(파일이 없으면 오류가 발생한다.)
w(쓰기) 파일에 데이터를 쓴다.
(파일이 있으면 기존 데이터는 삭제
되고, 파일이 없으면 생성된다.)
a(추가) 파일에 데이터를 이어 쓴다.
(파일에 있던 데이터는 유지된다.)
open( ) 함수의 파일입출력 모드에 따라 사용할 수 있는 파일 처리 함수가 정해진다.
보충 자료
파일입출력 모드 파일 처리 함수 특징
r(읽기) 파일 변수.read( ) 파일 전체 내용을 읽는다.
파일 변수.readline( ) 파일 포인터 위치부터 한 줄을 읽고,
파일 포인터의 위치를 다음 줄로 이동한다.
파일 변수.readlines( ) 파일 포인터 위치 이후의 모든 줄을 읽어서
각 줄을 요소로 갖는 리스트로 반환한다.
w(쓰기)
a(추가) 		파일 변수.write( ) 파일 포인터의 현재 위치에 내용을 작성한다.
파일 처리가 끝난 후에는 close( ) 함수를 사용해 닫아 준다.
파일 변수.close( )
파일 포인터
파일 포인터는 파일에서 입출력 동작이 발생하는 위치(커서가 깜빡이는 위치)를 나타낸다.
