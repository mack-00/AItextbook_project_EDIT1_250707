함수의 개념과 정의

1 함수
특정한 기능을 수행하는 명령어들을 하나로 묶어서 이름을 붙인 명령어의 집합이다.
반복되는 코드를 줄여 프로그램의 가독성을 높인다.
같은 코드를 여러 번 작성하지 않고 재사용하므로 프로그래밍 효율성이 향상된다.
2 함수의 정의
함수를 정의할 때는 함수의 이름을 정하는데, 함수의 기능을 잘 파악할 수 있도록 정하는 것이 좋다.
함수의 입력값을 받는 변수를 매개 변수라 하며, 함수가 외부에서 값을 전달받아 기능을 실행해야
할 때 매개 변수를 정의한다.
함수의 출력값을 반환값이라 하며, 출력값이 있을 때만 반환값을 정의한다.
정의된 함수를 사용하는 것을 함수의 호출이라 하며, 호출할 때 함수에 전달하는 값을 인수라 한다.
함수는 특정 작업을 수행하는 명령어의 집합으로,
이를 정의할 때는 함수 이름, 매개 변수, 반환값을 지정한다.
