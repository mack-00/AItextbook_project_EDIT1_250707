문제 이해

  문제 이해는 해결하고자 하는 문제를 이해하고, 컴퓨팅 시스템으로 문제를 해결할 수 있는지 판단하는 단계이다. 컴퓨팅 시스템으로 문제를 해결할 수 있는지 판단하기 위해서는 문제 상태와 수행 과정을 명확하게 설명할 수 있어야 한다.
문제 상태와 수행 과정 예시
문제 상태
문제 상태는 문제의 초기 상태, 현재 상태, 목표 상태를 말한다.
  • 초기 상태 문제가 발생한 최초의 상태
  • 현재 상태 문제가 해결되면서 변화되고 있는 상태
  • 목표 상태 문제가 해결된 상태
수행 과정
수행 과정은 초기 상태에서 목표 상태에 도달하기 위해 수행해야 할 작업들을 말한다.
