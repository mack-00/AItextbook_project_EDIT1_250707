사물 인터넷 시스템 구성

  사물 인터넷 시스템은 디바이스(Device), 네트워크(Network), 플랫폼(Platform), 서비스(Contents 또는 Service)로 구성된다.
사물 인터넷 시스템 구성
사물 인터넷 시스템 구성 설명
디바이스(D) 정보를 주고받는 사물이다. 디바이스는 센서, 액추에이터, 통신 모듈 그리고 이것을 제어하는 마이크로컨트롤러 등이 포함된다.
네트워크(N) 디바이스와 플랫폼을 연결하는 역할을 한다. 센서가 수집한 데이터를 전송하거나 사물들이 서로 정보를 교환하도록 유무선 통신을 통해 각종 사물들을 연결해 주는 매개체이다.
플랫폼(P) 디바이스에서 수집한 데이터를 저장하고 분석하는 역할을 한다.
서비스(C 또는 S) 사물에 부착된 센서로 수집한 정보를 플랫폼에서 소통하고 융합하여 사용자에게 맞춤형 서비스를 제공하는 역할을 한다.
사물 인터넷 시스템 구성 예시

스마트 홈 시스템은 다양한 센서로부터 수집한 데이터를 처리하며, 그 결과를 토대로 기기들을 제어함으로써 편리하고 안전한 주거 환경을 조성한다.

스마트 홈에서 다양한 센서와 모터의 역할을 설명하는 내용이다. 조명에는 인체 감지 센서가 있어 사람의 움직임을 감지하여 자동으로 조명을 켜거나 끄는 역할을 하고, 화분에는 토양 수분 센서가 있어 적절한 물 공급 시기를 알려준다. 자동차에는 거리를 측정하는 초음파 센서가 있어 자동차와 근접된 물체가 있는지 알 수 있다. 냉장고에는 온습도 센서가 있어 냉장고 속 온도를 일정하게 유지해 준다.
인체 감지 센서

사람의 움직임을 감지하는 인체 감지 센서를 사용해서 전등을 켜거나 끈다.

토양 수분 센서

토양의 수분을 측정하는 토양 수분 센서를 사용해서 식물을 관리한다.

초음파 센서

거리를 측정하는 초음파 센서를 사용하면 자동차와 근접된 물체가 있는지 알 수 있다.

모터

속도와 회전 방향을 제어할 수 있는 모터를 사용하여 선풍기를 회전하거나 무선 조종 자동차를 이동시킬 수 있다.

온습도 센서

자동으로 온습도를 측정하여 에어컨을 켜거나 끈다. 또한 냉장고의 온도를 일정하게 유지한다.

사물과 사물, 사물과 인터넷을 연결해 주는 기술에는 RFID, NFC, Beacon, Wi-Fi 등이 있다.

RFID(알에프아이디)

무선 주파수를 이용하여 사물을 식별할 수 있게 하는 기술이다.

예시 재고 관리, 하이패스
NFC(엔에프씨)

전자 기기들이 근거리 무선 통신을 할 수 있는 기술이다.

예시 금융 결제, 교통 카드, 스마트 홈
Beacon(비콘)

블루투스 기반으로 근거리에서 데이터를 전송하는 무선 통신 기술이다.

예시 박물관, 전시회
Wi-Fi(와이파이)

전자 기기들이 무선 통신할 수 있게 하는 기술이다.

사물 인터넷 시스템의 플랫폼은 산업 및 가정 환경에서 센서와 디바이스로부터 데이터를 수집하고, 클라우드에서 분석하여 자동화된 제어와 최적화된 서비스를 제공한다.

산업용 클라우드 컴퓨팅 IoT 플랫폼

인공지능 기술을 융합하여 디바이스에서 생성된 데이터를 클라우드에 수집하고 분석한다.

스마트 홈 IoT 플랫폼

스마트폰과 연결하여 집 안의 전등, 전자 기기 제어, 전력 제어, 검침 등이 가능한 서비스를 지원한다.

사물 인터넷 시스템의 서비스는 스마트 의류, 스마트 자동 사료 급식기와 같은 디바이스뿐만 아니라 스마트 결제, 스마트 쇼핑과 같은 디지털 서비스도 포함한다.

스마트 의류

다양한 센서들을 옷에 부착하여 호흡, 체온 등
여러 생체 신호를 얻어 이를 활용하거나 사용자의
몸을 안전하게 보호하는 역할을 한다.

스마트 자동 사료 급식기

스마트 자동 사료 인식 서비스는 반려동물을
효율적으로 관리하기 위하여 카메라와
이미지 인식 기술을 활용하여 동물이 얼마나
많은 양의 사료를 섭취했는지 파악한다.

스마트 결제

스마트폰, 태블릿, 스마트워치 등에 신용 카드나
계좌 번호 등의 결제 정보를 등록해 비밀번호
입력 또는 지문 인식만으로 결제하는 방식이다.

스마트 쇼핑

매장에 들어가면 고객의 위치와 정보를 확인하고
간단한 터치 동작만으로도 상품 검색,
자동 결제가 되는 방식이다.
