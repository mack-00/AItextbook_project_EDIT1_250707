데이터 시각화의 개념과 종류

  일상생활에서 발생하는 수많은 데이터는 그 자체만으로는 전체를 파악하기 어려운 경우가 많다. 데이터 시각화는 수치나 통계 데이터를 그래픽으로 표현하여 분석 결과를 직관적으로 이해하기 쉽게 만드는 기술이다. 데이터 시각화의 종류는 다음과 같이 다양하다.
  • 박스 플롯
  • 히스토그램 분포도
  • 산점도
  • 차트
데이터의 특징을 파악할 수 있는 데이터의 통계 분포와 이상치를 시각화한 것이다.
  • 정형 데이터로 얻어 낸 통계적 수치를 박스 모양으로 표현하여 시각화
  • 데이터 집합의 범위와 통계치를 시각적으로 확인할 수 있고, 이상치 발견에 용이함.
수치형 변수를 구간으로 나누어 각 구간에 속하는 데이터의 분포를 시각화한 것이다.
  • 가로축에 정량화된 특정한 간격(폭)을 표시한 후 각 구간 대응 값의 빈도를 세로축으로 시각화
  • 데이터 속성값이 전체적으로 어떻게 분포되었는지 볼 때 효과적임.
수치형인 두 개의 변수 간의 관계를 시각화한 것이다.
  • 좌표에 데이터의 속성값을 점으로 시각화
  • 데이터 속성 사이 관계를 확인할 때 효과적임.
수치형인 두 개의 변수 간의 관계를 시각화한 것이다.
  • 원형의 여러 부채꼴 조각으로 시각화
  • 전체에서 각 속성의 비율을 비교할 때 효과적임.
