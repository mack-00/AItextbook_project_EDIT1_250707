데이터 수집 방법

웹 크롤링

  웹 사이트의 정보를 조직적 또는 자동화된 방법으로 수집하는 프로그램을 웹 크롤러(web crawler)라고 하며, 웹 크롤러로 하는 데이터 수집 작업을 웹 크롤링(web crawling)이라고 한다. 웹 크롤링을 활용하면 웹 사이트에 있는 특정 정보를 자동으로 수집할 수 있다는 장점이 있다.
웹 크롤링을 수집할 수 있는 대표적인 사이트 예
ScrapeStorm
  • 인공지능 기술을 기반으로 개발된 무료 웹 데이터 수집 사이트이다.
  • 웹 사이트: https://kr.scrapestorm.com
Octoparse
  • 웹 사이트에서 필요한 데이터를 자동으로 추출하고 수집할 수 있는 사이트이다.
  • 웹 사이트: https://www.octoparse.com
