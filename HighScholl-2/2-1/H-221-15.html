2-1. 빅데이터 수집

1
답변을 이 곳에 놓으세요
: 기존의 관리 방법이나 분석 체계로는 처리하기 어려운 막대한 양의 데이터셋 또는 그것을
처리하는 기술
2 빅데이터의 특징
데이터의 크기
데이터 집합의 크기가 테라바이트에서 페타바이트에 이르기까지 매우 큼.
데이터의 생성 속도
센서, 인터넷, 모바일 장치를 포함한 다양한 곳에서 데이터가 빠르게 생성되고 수집됨.
종류 및 형식의 다양성
구조화된 데이터(데이터베이스), 구조화되지 않은 데이터(이메일, SNS 게시물) 등의 다양한 형태로 제공됨.
3 데이터 수집 방법
답변을 이 곳에 놓으세요
  • 공공 기관이 일상적 업무 수행의 결과물로 생성하거나 수집한 다양한 형태의 데이터
  • 개인 정보 보호 및 국가 안보와 질서를 위반하지 않으면 누구나 자유롭게 쓸 수 있도록 공개하고 있음.
민간 데이터
  • 주로 상업적 목적과 업무 효율성 향상을 위해 수집되는 다양한 형태의 데이터
  • 공공 데이터에 비해 좀 더 전문적이고 일상생활에 도움이 되는 데이터가 많아 활용 가치가 높음.
답변을 이 곳에 놓으세요
  • 웹 크롤러로 하는 데이터 수집 방법
  • 웹 사이트에 있는 특정 정보를 자동으로 수집할 수 있음.
답변을 이 곳에 놓으세요
  • 특정 목적을 위해 데이터를 처음부터 모으는 방법
  • 조사자에게 필요한 데이터가 없을 때 여러 센서 등을 활용하여 정확하게 데이터를 얻을 수 있음.
