디지털 데이터 압축의 개념

   디지털 기술의 발달로 전 세계에서 많은 양의 디지털 데이터가 만들어지고 있어, 데이터를 효율적으로 다루는 방법인 데이터 압축에 대한 관심이 높아졌다. 데이터 압축은 원본 데이터를 더 작은 용량의 데이터로 만드는 것으로 데이터의 불필요한 부분을 제거하거나 반복되는 부분을 효율적으로 표현하여 데이터의 크기를 줄이는 것이다.
디지털 데이터
컴퓨터는 여러 형태의 정보를 0과 1의 숫자로 변환하여 저장하는데, 이처럼 일련의 숫자 형태로 변환한 정보를 디지털 데이터라고 한다.
일상생활 속 데이터 압축의 예
음악 감상
데이터 압축은 고음질의 음악을 언제 어디서나 감상할 수 있게 해 준다. 소리 파일로 주로 사용되는 FLAC, MP3 등은 데이터 압축이 적용되어 있다.
영상 시청
데이터 압축은 고화질의 영상을 끊김 없이 시청할 수 있게 해 준다. 영상 파일로 주로 사용되는 MPEG, H.264 등은 데이터 압축이 적용되어 있다.
사진 촬영
데이터 압축은 스마트폰 등으로 간편하게 사진을  촬영하여 저장할 수 있게 해 준다. 이미지 파일로 주로 사용되는 JPEG, PNG 등은 데이터 압축이 적용되어 있다.
