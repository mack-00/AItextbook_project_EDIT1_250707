데이터 압축 방법

  데이터 압축 방법은 데이터의 손실 여부에 따라 무손실 압축과 손실 압축으로 나뉜다. 무손실 압축은 원본 데이터를 손실 없이 그대로 압축하는 방법이고, 손실 압축은 원본 데이터의 손실이 있는 압축 방법이다.

압축률
변환하기 전의 데이터와 변환 후의 데이터 크기 비율로서, 압축률이 높을수록 압축이 더 잘되었음을 의미한다.
압축률 =
원본 데이터 크기
압축된 데이터 크기
예를 들어 원본 데이터의 크기가 100KB이고, 압축된 데이터 크기가 80KB인 경우, 압축률은 1.25이다.
