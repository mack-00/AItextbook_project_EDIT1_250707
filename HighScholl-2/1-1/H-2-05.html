데이터 구조화

1 데이터 구조화
데이터 구조화: 데이터 간의 관계를 쉽게 파악하고 이해할 수 있도록 체계적으로 정리하고 표현한 것이다.
데이터를 구조화하면 데이터를 효율적으로 처리하고 관리할 수 있으며, 데이터 내용도 빠르게 전달할 수 있다.
2 데이터 구조화 방법
리스트
데이터를 나열하여 관리하는 구조화 방법
표
데이터를 행과 열을 기준으로 나열하는 방법
트리
노드로 데이터 간의 관계를
표현한 구조화 방법
그래프
점과 선을 통해 데이터를
표현하는 구조화 방법
데이터를 체계적으로 정리해 표현하기 위해서는 리스트나 표 등으로
데이터를 구조화하는 것이 필요하다.
