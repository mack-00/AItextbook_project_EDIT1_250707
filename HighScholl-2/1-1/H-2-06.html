데이터 분석

1 데이터 분석
구조화된 데이터를 바탕으로 데이터 간의 관계를 파악하고 데이터에 담긴 여러 의미를 해석하는 것을 의미한다.
데이터 분석은 데이터 수집, 데이터 전처리, 데이터 시각화의 단계로 이루어진다.
데이터 수집
분석에 필요한 데이터를
수집한다.
데이터 전처리
수집한 데이터를 분석에
적합한 형태로 가공한다.
데이터 시각화
가공된 데이터를 이해하기 쉽게 시각적으로 표현한다.
데이터 분석은 구조화된 데이터를 바탕으로 데이터 간의 관계 및 의미를 해석하는 것으로
데이터 수집, 데이터 전처리, 데이터 시각화 단계로 이루어진다.
